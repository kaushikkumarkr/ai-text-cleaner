# CleanText Pro

**The Ultimate Free AI Text Formatter & Word Counter**

CleanText Pro is a lightweight, frontend-only Single Page Application (SPA) designed to instantly clean out markdown formatting, fix spacing, and adjust casing for text generated by AI tools like ChatGPT, Gemini, and Claude. 

Perfect for users aiming to improve their productivity by easily sanitizing text before copying it into professional emails, essays, and documents.

## Features
- **Clean AI Formatting**: Instantly removes markdown asterisks, hashes, brackets, and unnecessary line breaks.
- **Fix Spacing**: Reduces double/triple spaces to single spaces and removes trailing whitespaces.
- **Sentence Case**: Standardizes casing, ensuring the first letter of every sentence is capitalized.
- **Real-Time Statistics**: Live word count, character count, and estimated reading time.
- **One-Click Copy**: Copy your sanitized text directly to the clipboard.
- **Privacy-First**: Operates 100% locally in your browser. No data is sent to a server.
